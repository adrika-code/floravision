# 🌿 FloraVision

A pixel-perfect, fully responsive plant e-commerce frontend built with **Next.js** and **Tailwind CSS** — developed as part of the **SoftDef Frontend Developer Internship Test**.

---

## 🖼️ Project Overview

FloraVision is a modern product listing page for a plant store, recreated faithfully from a provided Figma design. The project focuses on clean component architecture, responsive layouts, smooth interactivity, and accessibility — across all screen sizes from 320px to 1440px.

---

## ✨ Features

- 🧭 **Navbar** — Fixed navigation with logo, nav links, and mobile hamburger/drawer menu
- 🌄 **Hero Section** — Full-width banner with heading, subtext, and CTA buttons
- 🌱 **Trending Plants Section** — Highlighted plant cards with image, name, and price
- 🛒 **Top Selling Plants Section** — Product grid with hover effects
- 💬 **Customer Reviews Section** — Testimonials with avatar, name, and star rating
- 🌫️ **O₂ Plants Section** — Featured plant card with CTA
- 📬 **Footer** — Brand info, quick links, newsletter form, and social media icons
- 📱 **Fully Responsive** — Seamless layout across desktop, tablet, and mobile
- ♿ **Accessible** — Semantic HTML, alt text, focus indicators, keyboard navigation

## 🚀 Getting Started

### Prerequisites

- Node.js `v18+`
- npm or yarn

### Installation & Setup

```bash
# 1. Clone the repository
git clone https://github.com/your-username/floravision.git

# 2. Navigate into the project
cd floravision

# 3. Install dependencies
npm install

# 4. Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the app.

### Build for Production

```bash
npm run build
npm run start
```
